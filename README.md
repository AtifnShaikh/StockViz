# StockViz – Stock Analysis & Visualization System

## Description
StockViz is a Python-based stock analysis and visualization system that allows users to analyze real stock market data using simple natural language queries.

## Features
- Fetch current stock price
- Get today’s high and low price
- Compare two stocks
- Visualize stock trends using graphs
- Generate performance summaries

## Tools Implemented
1. Get Stock Price  
2. Get High & Low Price  
3. Compare Two Stocks  
4. Plot Stock Trend  
5. Stock Performance Summary  

## Libraries Used
- Python 3.14
- openai
- mcp
- yfinance
- matplotlib
- python-dotenv
